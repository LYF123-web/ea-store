# EA Store 部署到旧电脑指南

## 一、旧电脑环境准备

### 1. 安装 Node.js
```bash
# 下载 Node.js (推荐 v18 或更高)
# https://nodejs.org/zh-cn/
# 下载 LTS 版本安装
```

### 2. 安装 Git（可选）
```bash
# 下载 Git
# https://git-scm.com/downloads
```

## 二、复制项目文件

把整个 `mt5` 文件夹复制到旧电脑，包括：
- `mt5/src/` - 前端代码
- `mt5/server/` - 后端代码
- `mt5/package.json`
- `mt5/vite.config.js`
- `mt5/index.html`

## 三、在旧电脑上安装依赖

### 前端依赖
```bash
cd mt5
npm install
```

### 后端依赖
```bash
cd mt5/server
npm install
```

## 四、配置支付宝密钥

确保 `server/certs/` 目录下有两个文件：
- `app_private_key.pem` - 应用私钥
- `alipay_public_key.pem` - 支付宝公钥

## 五、启动服务

### 启动后端（端口3001）
```bash
cd mt5/server
node index.js
```

### 启动前端（端口5173）
```bash
cd mt5
npm run dev
```

## 六、局域网访问

如果只是局域网内使用：
1. 查看旧电脑的 IP 地址（如 192.168.1.100）
2. 其他电脑访问：`http://192.168.1.100:5173`

## 七、公网访问（让外网也能访问）

### 方法1：内网穿透（推荐，免费）
使用 frp、cpolar、natapp 等工具

以 cpolar 为例：
```bash
# 注册 cpolar 账号: https://www.cpolar.com
# 安装 cpolar
cpolar http 3001
# 会得到一个公网地址如: https://xxxx.cpolar.top
```

### 方法2：云服务器
1. 购买阿里云/腾讯云服务器
2. 安装 Node.js
3. 上传项目代码
4. 启动服务
5. 配置域名和 SSL 证书

### 方法3：端口映射
1. 登录路由器
2. 设置端口映射：外网端口 → 旧电脑IP:3001
3. 需要公网IP或配合 DDNS

## 八、设置开机自启（Windows）

创建 `start.bat` 文件：
```bat
@echo off
cd /d %~dp0\server
start node index.js
cd /d %~dp0
start npm run dev
```

把 `start.bat` 放到启动文件夹：
```
Win+R → shell:startup → 回车 → 把 start.bat 放进去
```

## 九、设置开机自启（Linux/Mac）

创建 `start.sh`：
```bash
#!/bin/bash
cd ~/mt5/server
node index.js &
cd ~/mt5
npm run dev &
```

添加到 crontab：
```bash
crontab -e
# 添加：
@reboot /bin/bash /path/to/start.sh
```

## 十、注意事项

1. **支付宝回调地址**：正式环境需要公网可访问的地址
2. **防火墙**：确保端口 3001 和 5173 没有被拦截
3. **安全**：生产环境建议使用 HTTPS
